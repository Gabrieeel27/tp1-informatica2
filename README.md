# tp1-informatica2
  Trabajo Practico numero 1 de Informática 2 de Gabriel Limongi del curso: 2-11R    -    
    Este proyecto consiste en una refacción a un programa dado en C y la idea era hacerlo denuevo pero utilizando funciones, este programa esta diseñado para gestionar el stock de un inventario de articulos de 3 sucursales, permiitendonos el cargar datos la automatizacion del conteo del stock total, el ordenamiento de mayor a menor mediante burbujeo y la visualizacion de los productos.

  Estructura, este codigo posee 3 archivos principales:
  
    funciones.h : Aqui se definen las constantes del sistema (la capcidad maxima de articulos, las sucursales), la estructura de los datos (articulos_t) y los prototipos de las funciones.
    
    funciones.c : Esta contiene la logica y el desarrollo de las funciones previamente declaradas.
    
    main.c : Aca esta el programa en si, la bienvenida, la inicializacion de estructuras y las llamadas a las otras funciones para luego estas lleven a cabo las tareas.
